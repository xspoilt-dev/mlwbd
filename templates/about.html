About MLWBD Movie Scraper

What is this?

MLWBD Movie Scraper is a web application that helps you search for movies and extract download links from MLWBD websites. It provides a clean, user-friendly interface to interact with movie databases.

Features

  • Movie search functionality
  • Multiple download options
  • Direct link extraction
  • Manual link processing
  • Responsive design
  • RESTful API endpoints

Technology Stack

  • Python Flask
  • Beautiful Soup
  • Requests
  • Bootstrap 5
  • JavaScript
  • HTML5

API Documentation

The application provides RESTful API endpoints for integration:

Description: Search for movies

Parameters:

  • q (required) - Movie search query

Example:

GET /api/search?q=avengers

Description: Get download links for a movie

Parameters:

  • url (required) - Movie URL

Example:

GET /api/links?url=https://example.com/movie

Description: Get direct download link

Parameters:

  • url (required) - Link URL

Example:

GET /api/direct?url=https://example.com/link

Installation & Usage

Prerequisites
  • Python 3.7 or higher
  • pip (Python package installer)
Installation Steps
  1. Clone the repository: 
    git clone https://github.com/xspoilt-dev/mlwbd.git
  2. Navigate to the project directory: 
    cd mlwbd
  3. Install dependencies: 
    pip install -r requirements.txt
  4. Run the application: 
    python app.py
  5. Open your browser and go to: 
    http://localhost:5000
